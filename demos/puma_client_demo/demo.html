🚀 PUMA Client 功能演示

体验本地客户端如何让网页拥有操作系统能力

💡 使用说明：请先运行 PUMA_Client-main/client.py 启动本地服务（端口 7175），然后刷新此页面。

👤 1. 获取当前用户信息

获取当前登录 Windows 的用户名（用于创建个人工作目录）

🔧 后端实现：
• getpass.getuser() - Python 标准库，获取当前系统登录用户名
• 无需额外安装，跨平台兼容 Windows/Linux/Mac
点击按钮查看结果...

📁 2. 打开本地文件夹

网页点击按钮，直接打开 Windows 资源管理器！这是普通网页做不到的事。

🔧 后端实现：
• subprocess.Popen(["explorer", path]) - 调用 Windows 资源管理器
• win32gui + win32con - Windows API，实现窗口强制前置
• win32process.GetWindowThreadProcessId() - 根据进程ID找到窗口句柄
• win32gui.SetForegroundWindow() - 将窗口置顶到最前
点击按钮查看结果...

📋 3. 复制路径到剪贴板

一键复制路径到 Windows 剪贴板，然后可以直接粘贴到任何地方

🔧 后端实现：
• os.system(f'echo {path} | clip') - 调用 Windows 的 clip 命令
• clip 是 Windows 内置命令，将文本复制到系统剪贴板
• 简单高效，无需第三方库
点击按钮查看结果...

📄 4. 扫描 Office 文件

扫描指定文件夹中的所有 Word、Excel、PowerPoint 文件

🔧 后端实现：
• os.listdir(folder_path) - 列出目录中的所有文件和文件夹
• os.path.isfile() - 判断是否为文件（排除文件夹）
• str.endswith(('.doc', '.docx', '.xls', '.xlsx', '.ppt', '.pptx')) - 过滤 Office 文件扩展名
• os.path.join() - 拼接完整文件路径
点击按钮查看结果...

🔗 5. 智能打开链接/文件

自动识别是网址、文件还是文件夹，并用正确的方式打开

🔧 后端实现：
• urllib.parse.urlparse() - 解析 URL，判断是 http/https/file 协议
• webbrowser.open() - 用系统默认浏览器打开网址
• os.startfile() - Windows 专用，用默认程序打开文件
• subprocess.Popen(["explorer", path]) - 打开文件夹
• platform.system() - 检测操作系统（Windows/macOS/Linux）
点击按钮查看结果...

📊 6. 获取项目信息（Protobuf）

从远程服务获取项目详细信息（需要有效的 UUID）

🔧 后端实现：
• requests.get() - 发送 HTTP GET 请求到远程 API
• pcminfo_pb2.ProjectProfile() - 创建 Protobuf 消息对象
• project_profile.ParseFromString() - 解析二进制 Protobuf 数据
• google.protobuf.json_format.MessageToDict() - 将 Protobuf 转为 JSON
• 数据定义在 pcminfo.proto 中，用 proto.py 编译生成 Python 代码
点击按钮查看结果...
